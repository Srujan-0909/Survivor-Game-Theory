# Survivor Strategy Simulator

A game theory project that models the strategic dynamics of the TV show **Survivor** using data analysis and simulation.

## Project Goal
Simulate a season of Survivor where players follow different strategic archetypes and attempt to maximize their probability of winning. The project aims to analyze strategies and eventually compute **Nash equilibria** for Survivor-style gameplay.

## Current Progress
- Analyzed historical Survivor data
- Built metrics such as:
  - Correct Vote Ratio
  - Survival Score
  - Target Rate
  - Immunity Wins
- Identified trends among winning players

## Next Steps
1. Define **player archetypes** (social player, strategist, challenge threat, floater).
2. Create **utility functions** representing each strategy.
3. Assume **Homo Economicus**: players maximize utility.
4. Build an **agent-based Survivor simulation**.
5. Run **Monte Carlo simulations** to estimate win probabilities.
6. Analyze **Nash equilibrium strategies**.
